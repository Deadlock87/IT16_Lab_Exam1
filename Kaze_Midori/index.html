How It Works

Caesar Cipher Formula

The Caesar Cipher uses a simple shift algorithm:

Encryption: E = (X + N) % 26

Decryption: D = (X - N) % 26

Where X is the position of the letter (A=0, B=1, ... Z=25) and N is the shift key.

Encryption Process

1. Combine inputs: FULL NAME | YEAR | COURSE

2. Shift each letter forward by N positions

3. Wrap around from Z to A

4. Keep spaces, numbers, and symbols unchanged

Security Features

• No eval() function used

• No unsafe DOM injection

• Input validation and sanitization

• Shift limited to 1-25 for security
