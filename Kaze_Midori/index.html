How Caesar Cipher Works

The Caesar Cipher is a substitution cipher where each letter in the plaintext is shifted a certain number of places down or up the alphabet. For encryption, each letter is shifted forward by N positions (E = (X + N) % 26). For decryption, each letter is shifted backward by N positions (D = (X - N) % 26). Numbers, spaces, and symbols remain unchanged. This creates a simple but effective encryption method that Julius Caesar reportedly used for military communications.
